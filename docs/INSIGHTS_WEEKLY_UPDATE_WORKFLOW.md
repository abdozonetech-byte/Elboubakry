# Insights Weekly Update Workflow

This page is weekly-update ready, but it does not publish automatically.

## Manual Workflow

1. Collect candidate topics from trusted marketing, AI, SEO, advertising, analytics and business sources.
2. Save only the topics that are relevant for Moroccan entrepreneurs, PME and brands.
3. Draft a short original insight with a practical angle.
4. Review manually for accuracy, usefulness, tone and personal brand fit.
5. Add the approved article to `light/assets/data/insights-articles.js`.
6. Update `publishDate`, `updatedDate`, `keywords`, `relatedTopics` and `sourceNotes`.
7. Review the page in desktop and mobile.
8. If individual article pages are generated later, update the sitemap and submit it in Search Console.

## Optional n8n Workflow Idea

1. Search latest marketing, AI, business, SEO, Meta Ads and growth topics.
2. Save candidate topics with source links and short notes.
3. Generate an original article draft from the notes.
4. Require human review before publishing.
5. Add only approved articles to the insights data file.
6. Rebuild and deploy the site.
7. Update sitemap after article pages exist.

No automatic publishing without review. No fake news. No scraping copyrighted articles. Summarize insights, verify facts, and write original content.
