# Checklist de verification SEO - Elboubakry Portfolio

## 1. Google Search Console

- Ajouter la propriete URL Prefix :
  https://abdozonetech-byte.github.io/Elboubakry/light/index-resume-one-page.html

- Methode recommandee :
  Balise meta HTML

- Remplacer dans le code :
  ADD_GOOGLE_SEARCH_CONSOLE_VERIFICATION_CODE_HERE

- Valider la propriete dans Google Search Console.

- Soumettre le sitemap :
  https://abdozonetech-byte.github.io/Elboubakry/sitemap.xml

## 2. Bing Webmaster Tools

- Ajouter le site :
  https://abdozonetech-byte.github.io/Elboubakry/light/

- Methode recommandee :
  Balise meta HTML

- Remplacer dans le code :
  ADD_BING_WEBMASTER_VERIFICATION_CODE_HERE

- Valider le site dans Bing Webmaster Tools.

- Soumettre le sitemap :
  https://abdozonetech-byte.github.io/Elboubakry/sitemap.xml

## 3. Verifications apres publication

- robots.txt accessible
- sitemap.xml accessible
- homepage indexable
- page a propos indexable
- page insights indexable
- articles indexables
- etudes de cas indexables
- aucune page publique en noindex
- URLs canoniques correctes
- donnees structurees JSON-LD valides
- GA4 installe avec le vrai Measurement ID

## 4. A faire lors du domaine final

Quand le domaine final sera connecte :

- remplacer les URLs canoniques
- remplacer les URLs dans sitemap.xml
- remplacer les URLs dans les schemas JSON-LD
- remplacer les URLs Open Graph
- ajouter une nouvelle propriete Google Search Console pour le domaine
- ajouter le domaine dans Bing Webmaster Tools
- soumettre le nouveau sitemap
